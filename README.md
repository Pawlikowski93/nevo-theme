# NEVO Theme

Custom block theme dla NEVO - Strategia. Technologia. Efekt.

## Wymagania
- WordPress 6.4+
- PHP 8.1+
- Node.js 20+

## Struktura
```
nevo/
├── assets/         # CSS, JS, images, fonts
├── blocks/         # Custom Gutenberg blocks
├── inc/            # PHP includes
├── parts/          # Template parts (header, footer)
├── patterns/       # Block patterns
├── templates/      # Page templates
├── functions.php   # Theme setup
├── style.css       # Theme header
└── theme.json      # Design system
```

## Development

### Lokalne uruchomienie
1. Clone repo do `/wp-content/themes/nevo`
2. Aktywuj motyw w WP Admin

### Build (później)
```bash
npm install
npm run build
```
